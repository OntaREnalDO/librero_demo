# Librero: The Future of Community Book Sharing 📚

Librero is a revolutionary startup demoing a circular economy platform for physical books. We connect readers through a smart locker system, making reading affordable, sustainable, and accessible.

## 🚀 The Vision
Librero aims to become the "Airbnb of Books." By leveraging local locker infrastructure, we eliminate the friction of physical meetups and traditional shipping, enabling a seamless "Drop and Go" exchange system.

## 💎 Core Features
- **Smart Locker Integration:** Every transaction provides a secure 3-digit access code and location.
- **Credit-Based Economy:** 
  - **Give to Get:** Earn 50 credits for every book you share.
  - **Affordable Rentals:** Rent any book for just 50 credits.
  - **Dynamic Ownership:** Buy books for 70-100 credits based on condition (Excellent/Good/Fair).
- **Partner Network:** Integrated advertising space for bookshops and universities, with credits redeemable for partner discounts.

## 📈 Business Plan & Profitability
- **Scalable Infrastructure:** Automated lockers mean zero staffing costs at exchange points.
- **Negative Working Capital:** Users purchase credits upfront (Starter pack: 200 credits for 1€ / 10 NOK), creating healthy cash flow.
- **Organic Inventory Growth:** Our reward system incentivizes users to provide the inventory, reducing platform acquisition costs to zero.
- **Revenue Diversification:** 
  - Credit sales.
  - B2B Partnership fees from bookshops/universities.
  - Premium subscription tiers.
  - Targeted advertising in the Partner Hub.

## 🛠️ Tech Stack
- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS (Modern, Responsive, Mobile-first)
- **Icons:** Lucide-React
- **State:** React Context API with LocalStorage persistence

## 🏃 How to Run the Demo
1. Ensure you have Node.js installed.
2. Run `npm install` (or `npm.cmd install` on Windows).
3. Run `npm run dev` to start the local development server.
4. Navigate to `http://localhost:5173`.
